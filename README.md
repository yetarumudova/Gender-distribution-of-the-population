# Gender-distribution-of-the-population Dashboard (Power BI)

# Project Overview
This project presents a comprehensive analysis of global demographic and social indicators using an interactive Power BI dashboard. The goal is to explore trends related to population distribution, gender differences, technology usage, and economic activity across different countries and years.

## Project Objectives
- Analyze gender distribution (male vs female population)
- Examine trends in births (boys vs girls)
- Investigate technology usage (phone & internet)
- Compare economic activity rates among women
- Explore transportation usage patterns
- Identify top countries based on population metrics

## Dataset Information
The dataset includes global-level data collected across multiple years and countries. The Dataset was taken from Databases of State Statistics Commite. It contains:
- Population by gender
- Birth statistics (male/female)
- Internet and phone usage rates
- Economic activity indicators
- Transportation usage data
- Country-level comparisons

Data format: Excel / CSV (processed and cleaned before visualization)

## Tools & Technologies Used
- Power BI – Data visualization & dashboard creation  
- Microsoft Excel – Data cleaning and preprocessing

## Key Insights

1.Gender Distribution
- Female population (~273K) is significantly higher than male population (~146K) in the dataset.

2.Birth Statistics
- Male births (~3M) are higher than female births (~2M), showing a consistent gender gap at birth.

3.Technology Usage
- Phone usage has steadily increased over the years, indicating digital growth.
- Internet usage also shows a gradual upward trend globally.

4.Country Analysis
- Certain countries dominate in population metrics (Top 10 visualization).
- Regional differences are clearly visible on the map.

5.Economic Activity for Women
- Female economic participation shows fluctuations, with a noticeable peak around 2021.

6.Transportation Usage
- Transportation usage varies by year, with 2019 showing the highest percentage.


##  Features of the Dashboard
- Interactive slicers (Year, Country)
- Multi-visual layout (bar charts, maps, pie charts, KPIs)
- Comparative analysis (male vs female)
- Time-series trends
